# 🦉 Curio

**Daily general-knowledge challenges, quick-fire quizzes and streaks.
Free forever. I am curious to become wise.**

Curio is a zero-dependency progressive web app: pure HTML/CSS/JS, no build
step, no backend, no tracking. Your streaks and scores live in your own
browser (localStorage). Install it from your phone's browser — it works
offline.

## Principles

1. All knowledge content is free, forever. No paywalled questions, ever.
2. No ad networks, no data selling, no engagement-bait dark patterns.
3. Every answer teaches: right or wrong, you get a "depth fact."
4. Works on a cheap phone in a browser. No forced app install.

## Run locally

Serve the folder with any static server, e.g.:

```
py -m http.server 8123
```

…or just open `index.html` in a browser (offline/PWA features need a server).
